## 🚀 خطوات التشغيل

### 1. فك الضغط
```bash
unzip hamada_ai_FINAL_v2.zip
cd hamada_ai_v2
```

### 2. تثبيت المكتبات
```bash
flutter pub get
```

### 3. توليد الكود (مهم!)
```bash
dart run build_runner build --delete-conflicting-outputs
```

### 4. تشغيل التطبيق
```bash
# على جهاز/محاكي متصل
flutter run --release

# أو build APK مباشرة
flutter build apk --release
# الملف في: build/app/outputs/flutter-apk/app-release.apk
```

## 🔑 أول إعداد

1. افتح التطبيق
2. اتبع شاشة الـ Onboarding
3. اذهب لـ **الإعدادات ← Groq API Key**
4. احصل على Key مجاني من: **https://console.groq.com**
5. الصق الـ Key واضغط "حفظ"
6. **حماده جاهز! 🎉**

## ✨ الميزات الجديدة في v2.0

### 🎤 Voice Input
- اضغط أيقونة الميكروفون في شاشة المحادثة
- تكلم بالعربي — حماده هيفهمك
- اضغط تاني لإيقاف التسجيل

### 💾 Backup & Restore
- **الإعدادات ← النسخ الاحتياطي**
- "تصدير": يعمل ملف `.zip` ويفتح Share Sheet
- احتفظ بالملف في Google Drive أو الميل
- "استرجاع": اختار ملف النسخة وحماده يرجّع كل حاجة

### 📊 تحليل مالي ذكي
- شاشة الفلوس ← تبويب "الحركات"
- اضغط "حلّل الشهر"
- حماده يحلل مصروفاتك ويديك توصيات شخصية

### 🎯 أهداف مالية
- شاشة الفلوس ← تبويب "الأهداف"
- اضغط + لإضافة هدف (شراء سيارة، إجازة، إلخ)
- اضغط ✚ على أي هدف لإضافة مبلغ

### 🔄 مصروفات متكررة
- شاشة الفلوس ← تبويب "المتكررة"
- اضغط + لإضافة إيجار / نتفليكس / راتب
- حماده يسجّلها تلقائياً كل فترة

### 🏠 Home Screen Widget
- اضغط على الشاشة الرئيسية لجهازك وأضف ويدجت
- ابحث عن "حماده AI"
- يظهر: المهمة الأهم + الرصيد الشهري + رسالة من حماده
